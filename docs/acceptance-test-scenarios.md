**作成日**: 2026-05-06

---

## 前提条件・制約事項

| 項目 | 内容 |
|---|---|
| 承認者の権限 | 承認操作（`approval:act`）はシステム管理者のみ保有。承認担当ユーザーはシステム管理者種別で作成する（将来的に権限設計を見直す予定） |
| マスタデータ | 得意先・仕入先・商品はAT実施前に投入済みとする |
| 多段承認 | 2段承認ルートのシナリオを含む（AT-02にて実施） |
| 印刷レイアウト | 確認対象外 |
